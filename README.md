# Superstore Analytics PowerBI Dashboard

A comprehensive and interactive Sales Dashboard built with Microsoft PowerBI to analyze Global Superstore sales data and derive meaningful business insights.

## 📊 Project Overview

This Superstore Analytics Dashboard provides deep insights into sales performance across different regions, categories, and time periods. The dashboard enables business stakeholders to make data-driven decisions regarding inventory management, sales strategies, and customer segmentation.

The project involves a complete ETL (Extract, Transform, Load) process using Power Query for data cleaning and transformation, followed by DAX (Data Analysis Expressions) for creating calculated measures and columns to support advanced analytics.

## 📁 Files Included

- **Global Sales Dashboard Project.pbix** - Main PowerBI dashboard file containing all visualizations, data models, and calculated measures
- **global_superstore_data.xlsx** - Raw dataset containing global superstore sales data
- **Sales Dashboard.png** - Screenshot of the dashboard visualization
- **README.md** - Project documentation

## 🚀 Getting Started

### Prerequisites
- Microsoft PowerBI Desktop (free version available for download)
- Microsoft Excel (for viewing the raw data)

### How to Use

1. **Download the PowerBI File**
   - Download `Global Sales Dashboard Project.pbix`
   - Open it with PowerBI Desktop

2. **View the Dashboard**
   - The dashboard will load with pre-configured visualizations
   - Use slicers and filters to interact with the data
   - Explore different metrics like sales, profit, and performance indicators

3. **Analyze the Raw Data**
   - `global_superstore_data.xlsx` contains the complete dataset
   - You can connect this Excel file to PowerBI for custom analysis
   - The data includes order details, customer information, product categories, and regional data

## 📈 Dashboard Features

### Key Metrics Tracked
- **Total Sales** - Overall revenue generated across all regions
- **Total Profit** - Net profit margin analysis
- **Sales by Category** - Performance breakdown by product categories
- **Regional Analysis** - Geographic distribution of sales and profits
- **Time-based Trends** - Monthly, quarterly, and yearly sales patterns
- **Customer Segmentation** - Analysis by customer segments and types

### Interactive Elements
- **Slicers** - Filter data by region, category, segment, and time period
- **Drill-down Capabilities** - Navigate from high-level summaries to detailed views
- **KPI Cards** - Key performance indicators at a glance
- **Charts and Graphs** - Visual representations including bar charts, line charts, and maps
